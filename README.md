## Optimaler Standort für eine Elektro-Motorrad Messe im Kanton Zürich 

Analyse und Visualisierung potenzieller Standorte für eine Elektro-Motorradmesse im Kanton Zürich:
- Integration und Bereinigung von Open-Data-Datensätzen (Energyreporter, Gemeindeporträt, GeoJSON)
- Entwicklung eines Ranking-Algorithmus zur Ermittlung der Zielgruppenrelevanz der Gemeinden
- Erstellung von Heatmaps und detaillierten Analysen der Top-Gemeinden
- Implementiert in Python mit Jupyter Notebooks
- Fokus: Datenaufbereitung, Analyse, Visualisierung und Entscheidungsunterstützung

Ziel: Ableitung datenbasierter Insights zur Standortwahl und zur zielgerichteten Ansprache vo
